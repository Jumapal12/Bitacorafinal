¿Para que esta hecho este proyecto?

Este proyecto esta hecho para aquellas personas que sufren del transtorno de autismo. Ayuda a poder tener una mejor comunicación de los padres, psicologos o doctores que traten con comportamientos o reacciones fuertes propios de las personas que sufren de este transtorno.

Problemática

Las personas con Autismo tienen dificultades para manejar y expresar sus emociones de forma convencional por lo que, puede parecer que reaccionan desproporcionadamente o “fuera de lugar”

Solución

Crear un producto especializado en el aprendizaje y reconocimiento de las emociones para personas autistas, este producto les ayudara a expresar las emociones que sientan.
